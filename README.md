# Scrabble Calculator (Laravel)


## MacOS
### Prerequisites
NodeJS v18

### Deploy Steps
Using terminal, in project root folder, run below commands to prepare the frontend.

1) Install dependencies 

        npm install

2) Start the app

        npm start

## Angular Tests
To generate test cases results for Angular, run the below command
        
        ng test

## Windows

1) Install dependencies 

        npm install

2) Start the app

        npm start

## Angular Tests
To generate test cases results for Angular, run the below command
        
        ng test
